# Simple Login Page

This is a simple Login Page project created using basic web technologies.  
The page allows users to enter their username and password to simulate a login interface.

## 🚀 Features
- Simple and clean login UI
- Username and password input fields
- Basic form validation
- Responsive design
## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
